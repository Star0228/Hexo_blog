hexo第一次安装及更新步骤

发表于 2024-02-20

1. 创建 GitHub Pages

  • Create a new repository：Repository name：<username>.github.io

创建完成，则博客地址为 https://<username>.github.io，如果 GitHub 账户名中有大写字母，地址会变成小写字母

2. 安装 Hexo

为 Hexo 创建文件夹如 Blog，进入 Blog 文件夹，在地址栏输入 cmd，则打开的命令行直接为当前目录。默认node.js 在16版本以上

  • 使用 npm 命令安装 Hexo，依次输入：

2.1 Hexo 常用命令

1.hexo-cli

  • 安装Hexo

2.hexo init

  • 初始化博客，名称为 blog，也可以换成其他名字

3.hexo generate

  • 生成

4.hexo sever

  • 启动服务预览
  • If you want to change the port or if you’re encountering EADDRINUSE errors, use the -p option to set a different port.

5.hexo deploy

  • 部署

6.hexo new

  • layout 可选参数，用以指定文章类型，若无指定则默认由配置文件中的 default_layout 选项决定

  • title 必填参数，用以指定文章标题，如果参数值中含有空格，则需要使用双引号包围

2.2 Hexo 目录结构

  • _config.yml：为网站配置参数，比如下面的部署就是在这里
  • source/_posts：你写的文章都在这里
  • themes：为博客设置主题
    -默认landscape
    -更换其他的主题步骤：
    将 _config.yml 内的theme修改为next <theme-name>即可
